# Movie-review-system
Movie Info Extractor is a lightweight, AI-powered Streamlit application that transforms unstructured movie descriptions into clean, structured data. Users simply paste a paragraph about a film — a synopsis, review, or news snippet — and the app extracts key details automatically using a large language model.
Under the hood, the system uses LangChain with Mistral's mistral-small-2506 model, guided by a Pydantic schema (Movie) that defines exactly what to pull out: title, release year, genre, director, cast, rating, and a short summary. A PydanticOutputParser ensures the model's response is parsed into this structured format rather than returned as loose text, making the output reliable and predictable.
The interface itself is designed to feel polished rather than utilitarian. Once the extraction completes, the data is rendered as a dark-themed "movie card" — the title and year appear prominently, genres show as pill-shaped badges, and fields like director, cast, and rating are neatly labeled. For users who want the raw output, a collapsible "View raw JSON" section reveals the exact structured data behind the card.
The workflow is intentionally simple: paste text, click "Extract," and view results — no unnecessary steps or clutter. If parsing ever fails, the app gracefully falls back to displaying the model's plain-text response, ensuring the user always gets an answer. It's a practical demonstration of combining structured LLM outputs with a clean, readable UI.
